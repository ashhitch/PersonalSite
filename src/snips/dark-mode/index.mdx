CSS TIP: Mac Dark Mode media queries (currently in Safari Tech Preview)

```css
/* Default Light Mode */
body {
  color: #333;
  background-color: #fff;
}

/* Dark Mode */
@media (prefers-color-scheme: dark) {
  body {
    color: #fff;
    background-color: #333;
  }
}
```
